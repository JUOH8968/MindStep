# 🚀 MindStep (miniproject_v6) 향후 개선사항 및 발전 방향

현재 MindStep 프로토타입은 훌륭한 시스템 아키텍처와 본질적인 AI 코칭 기능을 보여주고 있습니다. 향후 실제 사용자들이 매일 꾸준히 사용하는 '습관 형성 서비스'이자 완성도 높은 프로덕트(Product)로 발전하기 위해 다음 세 가지 핵심 업그레이드를 계획하고 있습니다.

---

## 1. 🧠 RAG (Retrieval-Augmented Generation) 도입을 통한 장기 기억 코칭

*   **현재 상태 (AS-IS):** 
    - 챗봇이 최근 10번의 대화 내용만 기억하는 구조(단기 기억 제한)로, 유저의 과거 다짐이나 고민을 오랫동안 기억하지 못합니다.
*   **개선 목표 (TO-BE):** 
    - **Vector DB**(Pinecone 등)를 도입하여 사용자와의 모든 대화 히스토리와 완료된 Task 내역을 텍스트 임베딩(Vectorize)해 저장합니다.
    - 새로운 대화 시 가장 연관성 높은 과거 맥락을 검색(RAG 기법)하여 프롬프트에 주입합니다. 이를 통해 *"지난달에도 비슷한 일로 힘들어하셨지만 결국 멋지게 해내셨잖아요."* 와 같이 **유저의 전체 히스토리를 이해하는 진정한 나만의 전담 코치**로 발전시킵니다.

## 2. � 모바일 앱(App) 전환 및 외부 캘린더 연동 (Google Calendar)

*   **현재 상태 (AS-IS):** 
    - React 기반의 웹(Web) 환경으로 구축되어 있으며, 자체 DB 내에서만 Task(할 일) 데이터를 관리하는 닫힌 구조입니다.
*   **개선 목표 (TO-BE):** 
    - React Native나 Flutter 등을 활용하여 **네이티브 모바일 애플리케이션**으로 포팅하여 접근성을 높입니다.
    - **Google Calendar API**(또는 Apple 캘린더)를 연동하는 Oauth 2.0 구조를 구축합니다. 챗봇을 통해 추출된 일정이 내 폰 캘린더에 즉시 동기화되고, 반대로 캘린더에 적힌 기존 일정을 챗봇(AI)이 읽어와서 코칭의 참고 자료로 쓸 수 있는 **스마트 스케줄링 생태계**를 완성합니다.

## 3. 🎯 행동 데이터 기반 주간/월간 리포트 (Personalized Analytics)

*   **현재 상태 (AS-IS):** 
    - 업무 완료 시 즉각적인 텍스트 피드백(칭찬/독려)은 제공되지만, 사용자가 자신의 패턴을 시각적으로 파악할 수 있는 통계 기능은 없습니다.
*   **개선 목표 (TO-BE):** 
    - DB에 쌓인 `action_log`(수정 방치, 완료율 등)와 챗봇 감정 데이터를 분석하여, 매주 일요일 저녁 **"주간 성장 리포트" 대시보드**를 제공합니다.
    - *'이번 주는 집중력 교관 페르소나와 함께 성취율이 15% 올랐어요!', '가장 많이 미루는 시간대는 밤 10시네요'* 등의 시각화 차트(Chart.js 등 활용)와 AI 종합 피드백을 통해 단순 툴(Tool)을 넘어선 **메타인지 향상 플랫폼**으로 고도화합니다.
