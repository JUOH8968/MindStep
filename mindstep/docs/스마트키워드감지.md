# 스마트 키워드 감지 기능 설명서

## 🎯 개선된 동작 방식

### ✅ 스마트 감지 (현재 구현)

사용자가 "다했어", "완료", "끝" 등의 키워드를 입력했을 때:

1. **남은 Task 개수 확인**
2. **남은 Task가 0개** → 넛지 중지 ✋
3. **남은 Task가 1개 이상** → 넛지 계속 ▶️

---

## 📋 시나리오별 작동

### 시나리오 1: Task 3개 있을 때

```
현재 Task:
- Task 1: 보고서 작성 (시작)
- Task 2: 회의 준비 (대기)
- Task 3: 이메일 답장 (대기)
```

#### 사용자: "다했어"
```
→ 남은 Task 3개 확인
→ 넛지 계속 진행 ✅
→ Task 2, 3에 대한 넛지 계속 나옴
```

**uvicorn 로그:**
```
[스마트 감지] 키워드 감지되었지만 남은 Task 3개 → 넛지 계속
```

---

### 시나리오 2: Task 1개만 남았을 때

```
현재 Task:
- Task 3: 이메일 답장 (시작)
```

#### Task 3 완료 처리
```
→ 남은 Task 0개
→ 자동으로 축하 메시지 + 넛지 중지 🎉
```

---

### 시나리오 3: 마지막 Task 작업 중 "다했어" 입력

```
현재 Task:
- Task 3: 이메일 답장 (시작) ← 아직 완료 안 함
```

#### 사용자: "다했어"
```
→ 남은 Task 1개 확인 (Task 3)
→ 넛지 계속 진행 ✅
→ "다했다면 완료 버튼을 눌러주세요" 같은 넛지 나올 수 있음
```

#### 사용자: Task 3을 "완료"로 변경
```
→ 남은 Task 0개
→ 모든 할 일 완료 메시지 + 넛지 자동 중지 🎉
```

## 🔧 구현 코드

### chat_task_api.py
```python
# 스마트 감지: 키워드가 감지되면 남은 Task 개수를 확인
should_stop_nudge = False
if keyword_detected:
    # 남은 Task 개수 확인 (완료되지 않은 Task)
    task_db_check = get_task_db()
    cur_check = task_db_check.cursor()
    cur_check.execute(
        "SELECT COUNT(*) as count FROM task WHERE User_ID=? AND status != '완료'",
        (user_id,)
    )
    remaining_tasks = cur_check.fetchone()["count"]
    task_db_check.close()
    
    # 남은 Task가 0개일 때만 넛지 중지
    if remaining_tasks == 0:
        should_stop_nudge = True
        print(f"[스마트 감지] 키워드 감지 + 남은 Task 0개 → 넛지 중지")
    else:
        print(f"[스마트 감지] 키워드 감지되었지만 남은 Task {remaining_tasks}개 → 넛지 계속")
```

## 💡 작동 흐름

```
사용자: "다했어" 입력
    ↓
키워드 감지 ("다했어" in stop_keywords)
    ↓
남은 Task 개수 조회 (DB: WHERE status != '완료')
    ↓
    ├─ 0개 → should_stop_nudge = True → 넛지 중지
    └─ 1개 이상 → should_stop_nudge = False → 넛지 계속
```

## 🧪 테스트 방법

### 테스트 1: Task 여러 개 있을 때
1. Task 3개 추가
2. "다했어" 입력
3. uvicorn 터미널 확인:
   ```
   [스마트 감지] 키워드 감지되었지만 남은 Task 3개 → 넛지 계속
   ```
4. 넛지가 계속 나오는지 확인 ✅

### 테스트 2: Task 1개씩 완료
1. Task 1개 "완료" 처리
2. 남은 Task 2개
3. "다했어" 입력
4. 넛지 계속 나옴 ✅

### 테스트 3: 모든 Task 완료
1. 마지막 Task "완료" 처리
2. 남은 Task 0개
3. "🎉 모든 할 일을 완료하셨습니다!" 메시지
4. 넛지 자동 중지 ✅

### 테스트 4: 키워드로 중지
1. 모든 Task 완료 (남은 Task 0개)
2. "다했어" 입력
3. uvicorn 터미널 확인:
   ```
   [스마트 감지] 키워드 감지 + 남은 Task 0개 → 넛지 중지
   ```
4. "수고하셨어요! 넛지 알림을 중지했습니다. 🎉" 메시지 ✅

## ✨ 장점

### 1. 사용자 친화적
- 실수로 "다했어" 입력해도 남은 Task가 있으면 넛지 계속
- 다 끝나지 않은 상태에서 넛지가 중지되는 불편 방지

### 2. 자동 완료 감지
- 마지막 Task 완료 시 자동으로 넛지 중지
- 수동으로 "다했어" 입력하지 않아도 됨

### 3. 명확한 피드백
- uvicorn 터미널에서 실시간 로그 확인
- 디버깅 용이
