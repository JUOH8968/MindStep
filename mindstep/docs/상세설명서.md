3. **`startNudgeTimer()` 함수 수정**
   ```javascript

   console.log(`넛지 타이머 시작 - ${settings.persona_type}: ${interval/1000}초 간격 폴링`);
   
   // 첫 넛지는 간격의 절반 후에 실행

   // 설정된 간격마다 넛지 폴링
   nudgeTimerRef.current = setInterval(pollNudge, interval);
   ```

#### ✨ 개선 사항
- **첫 넛지 시간 단축**: 간격의 절반 후 첫 넛지 (예: Type B는 2.5초 후)
- **콘솔 로그 개선**: 페르소나와 간격을 명확히 표시

```python
def is_do_not_disturb_time():
    """방해 금지 시간(23:00~05:00) 체크"""
    now = datetime.now()
    current_hour = now.hour
    
    # 23:00~23:59 또는 00:00~04:59
    if current_hour >= 23 or current_hour < 5:
        return True
    return False
```
